# Sitio Web Diagnosticar - Centro Médico

Este es un sitio web estático para un centro médico integral con especialidades y diagnóstico por imágenes.

## Estructura del Sitio

```
diagnosticar/
├── index.html                    # Página principal
├── servicios.html               # Listado de todos los servicios
├── sedes.html                   # Información de sedes
├── turnos.html                  # Sistema de turnos
├── resultados.html              # Portal del paciente - resultados
├── faqs.html                    # Preguntas frecuentes
├── assets/                      # Recursos estáticos
│   ├── css/                     # Archivos CSS
│   ├── js/                      # Archivos JavaScript  
│   └── images/                  # Imágenes
├── servicios/                   # Páginas detalladas de servicios
│   ├── cardiologia.html
│   ├── resonancia.html
│   ├── ecografia.html
│   ├── ginecologia.html
│   ├── traumatologia.html
│   ├── pediatria.html
│   ├── mamografia.html
│   ├── gastroenterologia.html
│   └── apto-fisico.html
├── sedes/                       # Páginas específicas por sede
├── portal-paciente/             # Funcionalidades del paciente
│   └── whatsapp.html           # Atención por WhatsApp
└── README.md                    # Esta documentación
```

## Características del Sitio

### Tecnologías Utilizadas
- **HTML5** semántico
- **Tailwind CSS** vía CDN para estilos
- **JavaScript vanilla** para interactividad
- **Responsive design** para todos los dispositivos

### Funcionalidades Principales
1. **Homepage** con información general y accesos rápidos
2. **Servicios médicos** organizados por especialidades
3. **Sistema de turnos** online y por WhatsApp
4. **Portal del paciente** para consultar resultados
5. **Información de sedes** con direcciones y horarios
6. **FAQ's** con preguntas frecuentes organizadas por categorías
7. **Atención por WhatsApp** con enlaces directos

### Servicios Incluidos

#### Especialidades Médicas
- **Cardiología**: ECG, Holter, Ergometría, Ecocardiograma
- **Ginecología**: Controles, PAP, Colposcopía
- **Traumatología**: Consultas, Radiografías
- **Pediatría**: Controles, Vacunación
- **Gastroenterología**: Consultas especializadas
- **Apto Físico**: Certificados deportivos y laborales

#### Diagnóstico por Imágenes
- **Resonancia Magnética**: Cerebro, Columna, Articulaciones, Abdomen
- **Ecografía**: Abdominal, Ginecológica, Obstétrica, Doppler
- **Mamografía**: Digital con tecnología avanzada

### Sedes
- **Berazategui**: Sede principal con todos los servicios
- **Lomas de Zamora**: Sucursal con servicios principales

### Coberturas
- **PAMI**: 100% sin copagos
- **IOMA**: 100% sin copagos  
- **Otras obras sociales**: OSDE, Swiss Medical, Galeno
- **Pacientes particulares**: Con facilidades de pago

## Consideraciones Técnicas

### Imágenes
- Las rutas de imágenes apuntan a `assets/images/`
- Se recomienda usar formatos modernos (AVIF, WebP)
- Incluir texto alternativo para accesibilidad

### SEO
- Títulos descriptivos en cada página
- Meta descripciones optimizadas
- Estructura semántica HTML5
- Schema.org para información médica

### Performance
- Tailwind CSS vía CDN para prototipado rápido
- Imágenes con carga lazy
- Estructura optimizada para carga rápida

### Accesibilidad
- Navegación por teclado
- Contraste de colores adecuado
- Texto alternativo en imágenes
- Estructura de encabezados lógica
